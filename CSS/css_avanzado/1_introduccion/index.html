Introduccion

que es una animacion/ transicion

las animaciones proporcionan una forma de intrpolar valores de propiedades CSS cuando cambian como resultado de cambios subyacentes en las propiedades. Cuando el estado inicial y el estado final de una propiedad sucede en un periodo de tiempo, se denomina interpolacion

fases del renderizado

cuando se cambia una propiedad el navegador tiene que recalcular como dibujar los elementos. Existe 3 fases a la hora de ver los elementos en pantalla.

  1. Layout: Como se colocan los elementos en pantalla (consume muchos recursos)
  2. Paint: Dibujar o pintar los pixeles de los elemento (consumo medio)
  3. Composite: Componer y dibujar los elementos en pantalla (consumo bajo)

Propiedades Animables

  • No todas las propiedades CSS son animables
  • Para animar una propiedad el navegador tiene que poder calcular estados intermedios
  • Todas las propiedades numericas CON ESTADOS INTERMEDIOS son animables: margin, paddin, font-size, color, etc
